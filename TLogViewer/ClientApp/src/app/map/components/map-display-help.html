Map display lines

Display heading

Shows a red line from the aircraft in the direction of its current yaw / heading (the way the nose is pointed).

Display target path

Shows an orange line in the navigation bearing / target direction — the path the autopilot is trying to follow toward the active setpoint.

Display wind

Shows a dark green line with an arrow for estimated wind direction. Line length scales with wind speed when that data is available in the log.

Flight trail colors

The trail behind the aircraft is colored by ArduPlane flight mode (HEARTBEAT customMode). Larger dots mark the moments when flight mode changes.
